Frugal learning

Exploring pre-training methods to improve downstream tasks

Introduction to Machine Learning

ID document classification plays a critical role in automated identity verification, helping businesses and institutions efficiently process passports, driver’s licenses, and other forms of identification. However, due to the highly sensitive nature of these documents and strict data protection regulations like the General Data Protection Regulation (GDPR), developing such systems requires careful attention to data privacy and frugality.

The main goal is to find the best pre-training methods that can enable model to learn the best image representation in an unsupervised manner and need very little annotation (Few-shot/one-shot) to perform well on any given downstream task.

Overview

To address the challenge of limited labeled data, self-supervised learning (SSL) techniques such as Masked Siamese Networks (MSN) and SwAV (Swapping Assignments between Views) from Meta AI provide promising solutions. These methods enable pre-training models on unlabeled or synthetically generated ID-like images, reducing reliance on sensitive real-world ID documents.

Achievements:

  • End-2-End pipeline for model training and evaluation on internal/public datasets.
  • Finetunning models on client's data.
  • Set up a new SOTA performance on client’s benchmark.
